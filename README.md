# 🚀 AI Self-Healing Test Automation System

## 📌 Overview
This project automates test-failure analysis using n8n and Google Gemini AI.

The workflow:
- Detects test failures
- Analyzes root cause using AI
- Suggests fixes automatically
- Simulates self-healing automation

---

## ⚙️ Tech Stack
- n8n
- Google Gemini API
- Webhooks
- JavaScript

---

## 🧠 Workflow Architecture

Webhook Trigger → Failure Extraction → Gemini AI Analysis → Self-Healing Logic

---

## 🚀 Features
✅ Automated failure detection  
✅ AI-powered root cause analysis  
✅ Suggested self-healing fixes  
✅ Modular workflow design
